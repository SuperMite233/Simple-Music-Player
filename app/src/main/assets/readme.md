# Simple Music Player

Simple Music Player (SMP) is an Android music player built with Kotlin.

It supports local music scanning, metadata reading, playlists, playback history, favorites, timed lyrics, CUE parsing, per-track aliases, comments, tags, ratings, and basic streaming playback.

Current streaming support includes DizzyLab login, purchased album loading, album detail parsing, streaming playback, and downloads. More streaming platforms will be supported in future versions.

This project uses ChatGPT-assisted development. The NCM conversion feature includes code adapted from NCMConverter4a.
