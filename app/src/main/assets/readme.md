# Simple Music Player

Simple Music Player，简称 SMP，是一款使用 Kotlin 开发的 Android 音乐播放器，面向本地音乐管理、歌单管理、播放控制和轻量流媒体播放场景。

当前支持自动扫描 Android 系统媒体库、手动导入音频/CUE/LRC 文件、授权文件夹扫描、读取标题、歌手、专辑、年份、时长、曲序、作曲家、时间戳和封面等音乐元数据。播放界面支持歌词按时间显示、封面手势切歌、正在播放队列、播放模式、倍速、音量和歌曲详细信息。

歌单方面，SMP 支持普通歌单、我喜欢的音乐、最近播放、播放排行、CUE 歌单和自动专辑歌单。用户可以为音乐添加别名、点评、标签和星级评分，并在音乐库中搜索或筛选。

外观方面，SMP 支持主题色、背景图片、浅色模式、深色模式和跟随系统。悬浮歌词可以设置显示位置、透明度，并使用主题色显示描边文本。

流媒体方面，当前已经支持 DizzyLab 登录、已购专辑加载、专辑详情解析、串流播放和下载。DizzyLab 下载会写入 `album.json` 专辑索引，后续扫描时优先使用索引中的专辑封面、曲目数量和曲目元数据。后续版本计划继续支持更多平台。

本项目使用 ChatGPT 辅助构建。音乐格式转换功能中的 NCM 解密核心参考并移植自 MIT 许可项目 NCMConverter4a。
