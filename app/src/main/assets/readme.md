# Simple Music Player

Simple Music Player，简称 SMP，是一款使用 Kotlin 开发的 Android 音乐播放器。

当前支持本地音乐扫描、音乐元数据读取、播放列表、播放历史、我喜欢的音乐、按时间显示 LRC 歌词、CUE 文件解析、单曲别名、点评、标签、评分和基础流媒体播放。

流媒体方面当前已经支持 DizzyLab 登录、已购专辑加载、专辑详情解析、串流播放和下载。后续版本会继续支持更多平台。

本项目使用 ChatGPT 辅助构建。NCM 转换功能的相关核心代码参考并移植自 NCMConverter4a。
